grapher.js



a micro js charting framework

current version: 0.35
5.4KB (2.23KB gzipped)

source 11.6KB
minified 5.4KB

grapher.js is a micro (very small) pure javascript framework for graphing various charts. Its goal is to be a self contained pure javascript library, that does not use canvas, is blazing fast, and supports more than modern browsers, with a tiny footprint and a pretty face. There are some awesome things about grapher besides its holistic approach to size:

. requires no external libraries(no jquery), no images, or css
. does NOT use canvas, no VML or SVG here.
. can be completely inlined (tiny)!
. is blazing fast!
. works practically everywhere!
. features easy to use syntax.
. is FREE , OPENSOURCE , and EVOLVING!

Absolutely no restrictions on how you can use grapher.js, but if you do use it, please drop me a line!

gallery and usage guide

Bar Graph, Horizontal Style, with Bar stack

 
 
 

usage

 
<script type="text/javascript"> 
	data = {"1":[100,50,20],"2":[98],"3":[75,40],"4":[60],"5":[70,36],"6":[30]}
	options =   {"borderColor":"#000",
    "width":"300px",
    "height":"150px",
    "ymax":800,
    "gridlines":true,
	"yunits":["100","90","80","70","60","50","40","30","20","10"],
	"xunits":["Mon","Tue","Wed","Thu","Fri","Mon","Tue","Wed","Thu","Fri"],
	"style":"h"
    }
	grapher.bar("test",data,options);
</script>
